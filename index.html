You would find the following table in your SQL online compiler:

Table: FUND_PERFORMANCE

Description:

Monthly performance of all funds.

Schema:

  • fund VARCHAR(500)
  • evaluation_year INT(10)
  • evaluation_month INT(10)
  • fund_return FLOAT(12)

Variable explanation:

  • fund : fund name
  • evaluation_year : the year which the fund performance data is captured
  • evaluation_month : the month which the fund performance data is captured
  • fund_return : monthly return of the fund

Table View:

fund evaluation_year evaluation_month return

Fund A 2021 3 0.28
Fund B 2021 1 -0.25
Fund B 2021 2 0.26
Fund B 2021 3 -0.05
Fund C 2021 1 0.12
Fund C 2021 2 -0.09
Fund C 2021 3 0.28
