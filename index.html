You would find the following 2 tables in your SQL online compiler:

Table 1: FUND_MANAGER

Description:

Details of fund manager for each fund.

Additional Details:

Each fund manager could only manage one fund.

Schema:

  • fund VARCHAR(500)
  • manager VARCHAR(500)

Variable explanation:

  • fund : fund name
  • manager : fund manager

Table View:

fund manager
Fund A Adam
Fund B Nicole
Fund C Alicia

Table 2: FUND_PERFORMANCE

Description:

Time series data of fund performance for each fund.

Schema:

  • fund VARCHAR(500)
  • evaluation_year INT(10)
  • evaluation_month INT(10)
  • return_1y FLOAT(12)

Variable explanation:

  • fund : fund name
  • evaluation_year : the year which the performance data is captured
  • evaluation_month : the month which the performance data is captured
  • return_1y : 1-year return of the fund

Table View:

fund evaluation_year evaluation_month return_1y

Fund A 2021 2 0.25
Fund A 2021 3 0.28
Fund B 2021 1 -0.25
Fund B 2021 2 0.26
Fund B 2021 3 -0.05
Fund C 2021 1 0.12
Fund C 2021 2 -0.09
Fund C 2021 3 0.28

Task:

You would need to design a SQL query which tells which fund manager(s) generate(s) the best return on 2021 March.

Expected Output:

manager
Adam
Alicia
